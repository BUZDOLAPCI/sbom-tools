# Dedalus Marketplace Info

## Short Description
Create, merge, and diff Software Bills of Materials (SBOMs)

## Long Description
Generate CycloneDX or SPDX SBOMs from dependency lists, merge multiple SBOMs into one, and diff SBOMs to find added, removed, or changed components. Tools: sbom_from_dependencies, sbom_merge, sbom_diff.

- Code
- DevOps
